Place tile PNG textures in this folder. Expected filenames:

- grass.png
- stone.png
- rock.png
- water.png
- bridge.png
- bush.png
- tree.png
- tallGrass.png
- house.png
- hut.png
- dirt.png
- portal.png

Notes
- These files are optional. If missing, the engine auto-generates placeholders so the game still runs.
- You can add more tile textures and reference them in your legend as `textureKey` values (e.g., `tiles/water`).
- If you add new textures, update your map legend (in loader or your map pack) to point to them via `textureKey`.
